# 따릉이 랜덤 디펜스

엑셀로 취합한 따릉이 보관소를 레버를 당기게 되면 랜덤으로 출력이 됩니다.

해당 보관소를 향해서 자전거를 타고 떠날 수 있는 서비스입니다.

## 실행 방법

```shell
yarn install
yarn dev
```

## 배포 URL

https://bike-trip-xi.vercel.app/
